## 📲 How to Install

1. Download the latest **APK** from the [Download Button](#) above.
2. Go to your Android `Settings` > `Security`.
3. Enable **"Install from Unknown Sources"**.
4. Open the downloaded `.apk` file and tap **Install**.
5. Open **DevEditor Pro** and Start Coding! 🚀
